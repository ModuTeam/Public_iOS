# 프로젝트 소개
링크모아는 소중한 링크들을 쉽게 찾아보고 정리할 수 있는 서비스입니다.   
다양한 주제에 맞게 링크를 정리하고 공유함으로써 지식을 쉽게 공유할 수 있습니다.
